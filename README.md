A student-first smart cooking companion built for the FoodoScope API Hackathon.

## Problem
Students struggle with deciding what to cook due to limited time, ingredients, and energy.

## Solution
CampusCravings uses FoodoScope's RecipeDB and FlavorDB APIs to provide quick,
personalized, and flavor-optimized meal suggestions.

## APIs Used
- RecipeDB
- FlavorDB
